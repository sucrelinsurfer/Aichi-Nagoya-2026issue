# 網站規劃：2026 名古屋亞運衝浪選手困境懶人包

> 版本：v0.1 規劃書（grill-me 訪談後定案）
> 日期：2026-07-07

---

## 一、定位與目標

**一句話定位**：一個「凡事查證」的亞運衝浪爭議懶人包——用情感與急迫性讓議題被看見，用逐句查證建立公信力。

**核心目標（依優先序）**

1. 讓議題「被看見、被理解」——時效壓力大（賽事九月登場，選手仍上不了場）。
2. 促使運動部給一個公開說法。
3. 示範好習慣：不要看到什麼就信什麼，每句話旁邊都有查證。

**調性原則**：情感與急迫性打頭陣（首頁倒數＋核心數字），但「每句主張都有查證標籤」是可信度骨幹與差異化，避免被一句「帶風向」擊倒。事實區與訴求區明確分開。

---

## 二、受眾與行動出口

**主要受眾**：一般大眾（靠分享製造輿論壓力）。
**次要對象**：運動部（直球要說法）。

**行動出口（頁面結尾）**

- **一鍵分享**：預設文案＋圖卡，一鍵發 FB／Threads／IG／LINE。
- **立場投票**（互動＋擴散主軸），四個選項：
  1. 取得國際配額，只要符合協會選拔，就應代表國家出賽。
  2. 即使取得配額，仍需再經國家競技門檻審查。（＝現行政府立場）
  3. 可以審查，但選手願意自費，應允許出賽。
  4. 新興運動應採專案審查制度，協會應積極提出選手競技實力與發展潛力等完整資料供審查，並將協會辦理選訓及行政績效列為年度補助的重要依據。

> 投票選項刻意保持平衡（含政府立場），呼應中立精神，同時驅動互動。

## 四、各頁內容重點

### 事件時間軸

九項時序逐一展開，每項掛查證標籤。時序與查證狀態已完成（見《亞運衝浪爭議_時序查證與來源對照.md》）。需注意的修正點：

- 配額為 **2024＋2025 兩屆**亞錦賽團體積分累加，非單一 2025 賽事。
- 男子「第 25 名」查無來源，中央社僅載最佳第 17 名 → 標為待查或移除。
- 「費用我出」出自財經作家陳重銘（第三方），非選手本人 → 歸屬要清楚。
- 2020/8、2022/9 法規修訂確切日期與附件2 原文尚無公開正本 → 待補。

### 配額怎麼算？（本站關鍵洞察）

**官方配額表**（ASF，2026/6/1 公告，總計男 24／女 24）：

| NOC | 男 | 女 |
|---|---|---|
| 中華台北 | 2 | 2 |
| 日本 | 1 | 2 |
| 印尼 | 2 | 2 |
| 韓國 | 2 | 2 |
| 泰國 | 2 | 2 |
| 馬爾地夫 | 2 | 2 |
| …（共 17 國） | | |

**核心解讀**：配額由「2024 & 2025 兩屆亞錦賽的**國家團體排名**」換算，看的是**有沒有派隊去累積團體積分**，不是國家整體實力。

**為何日本強卻男子僅 1 席？**（→ 標記為「推論，待查證」）

- 日本頂尖選手多在 WSL 世界巡迴賽，未回來打亞錦賽，團體積分偏低。
- 台灣兩屆都派滿隊，穩定累積團體積分。
- 亞運採總量管制、每國有席次上限，與亞錦賽（每國可派多人）競爭結構不同。

> 來源：[ASF 配額公告](https://asiansurfing.org/2026/06/quota-places/)、[中央社協會9點聲明](https://www.cna.com.tw/news/aspt/202607060348.aspx)、[聯合報家長投書](https://udn.com/news/story/7005/9580336)

### 參賽國一覽

各國遴選方式＋國情限制（宗教、經濟、無海岸線等），初稿已完成（見《亞洲各國衝浪國家隊_遴選標準總體檢.md》），逐條補查證標籤。

## 五、招牌功能：查證元件規格

**互動形式**：可展開「查證」標籤（inline chip）。點擊展開查證說明、來源連結、截圖。

**狀態分類（色塊）**

- 🟢 已證實
- 🟡 需修正／部分屬實
- ⚪ 待查／無獨立佐證
- 🔵 推論（標明為推測，非事實）

**每個查證含**：主張原文 → 狀態 → 查證說明 → 來源連結（原始＋archive.today 快照）→ 截圖（可點開放大 lightbox）。

**資料結構（claims.json 範例）**

```json
{
  "id": "quota-2year",
  "claim": "台灣以團體積分取得 2 男 2 女共 4 席",
  "status": "verified",
  "note": "配額由 2024＋2025 兩屆亞錦賽團體排名累加而成。",
  "sources": [
    {
      "title": "ASF 配額公告",
      "url": "https://asiansurfing.org/2026/06/quota-places/",
      "archive": "https://archive.today/xxxx",
      "screenshot": "/verifications/quota-asf.png"
    }
  ]
}
```

## 六、技術與部署

- **框架**：Next.js（App Router）＋ Tailwind CSS。
- **部署**：Vercel。
- **投票存票**：Vercel KV（或一支 serverless API route＋KV），前端即時顯示比例；以 cookie／localStorage 防重複投票（輕量防刷，非嚴格）。
- **內容管理**：時間軸與查證資料以 JSON／MDX 管理，方便持續增補。
- **RWD**：手機優先。

---

## 七、截圖與存證流程

1. 由我用瀏覽器工具逐一進入每個來源頁截圖，存成圖檔放進 `/public/verifications/`。
2. 每個來源同步存進 **archive.today** 永久快照，防止原文被刪改。
3. 截圖檔名對應 claim id，方便維護。

## 八、法律與可信度守則

- 只在「公務與公開發言」範圍內引用，據實、附原始出處。
- 不臆測個人動機、不加油添醋、不放私人資訊。
- 推論與事實明確分流（🔵 推論標籤）。
- 事實區與訴求區視覺分開。
